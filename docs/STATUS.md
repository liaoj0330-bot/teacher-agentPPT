# 进度与剩余风险

更新时间：2026-07-15

## 已完成并有自动化证据

- 服务器冻结版本读取和不可覆盖版本链。
- 过期基线提交返回 409。
- 教材附加、对话建议应用、页面修复均创建新版本。
- 版本化 PPTX 导出写入 `CoursewareArtifact`。

- 13 类教师课件版式协议。
- 页面 PASS / REVIEW / FAIL Gate 与单页重试。
- 视觉边界、重叠、字号、文本溢出和可编辑性检查。
- PPTX 模板解析与版式评分隔离 POC。

最近一次版本事实源 E2E：10/10 通过。

## 正在收口

- 无截图真人按钮验收：脚本已加入逐步落盘和逐页场景变化断言，完整流程仍需最终通过。
- 图片供应商：需要分别验证鉴权、超时、限流、真实图片响应和降级状态，不能用占位图冒充成功。
- 图片同源：生成图片需要绑定冻结版本，并由浏览器和 PPTX 场景共同消费。
- 模板主入口：模板解析已完成，模板选择、持久化及版本追踪仍需接入教师主工作区。
- Dashi 类工具仅允许做隔离 POC，许可证和导出实现不可直接混入主链。

## 已知运行风险

- 开发端口可能被遗留 Node 进程占用；启动前应检查 3002 健康状态。
- 当前是开发服务，不包含生产部署、监控、备份和容量结论。
- 没有真实供应商密钥时，AI 对话或图片接口可能进入明确的不可用/降级状态。

## 发布门槛

满足以下条件前，不标记为商业正式版：

1. 真人主流程所有关键按钮通过且无未处理 Promise、无静默失败。
2. 图片供应商真实调用和失败降级均有自动化证据。
3. 模板选择进入版本事实源。
4. 生产构建、迁移、冷启动和导出回归通过。
5. 仓库和部署环境完成密钥、上传材料及日志数据审计。
